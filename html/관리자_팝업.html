※ 엑셀 파일을 업로드 진행 단계

  • 엑셀파일 저장 시 엑셀 형식으로 저장하여 사용할 것(샘플파일 참조)
  • 엑셀파일 작성 시 [사번, 주민등록번호, 성명, 소속기관, 부서명, 직급명, 사내전화번호, 회원구분, 성별, 생년월일, 이메일, 휴대폰번호, 집주소, 상세주소, 우편번호, 메시지수신방법, 재직여부, 계열사]의 순서로 작성
  • 코드데이터는 샘플파일 참조Sheet를 참고하여 작성

※ 엑셀 파일 업로드 유의사항

  • 아이디는 사번으로 아이디가 등록 됩니다.
  • 파일 업로드를 통한 회원등록 시 비밀번호는 생년월일 뒤 6자리 (주민등록번호 앞 6자리)로 등록 됩니다.
  • 엑셀파일 작성 시 기등록된 사번을 입력할 경우 해당 회원은 등록되지 않습니다.
  • 엑셀파일 작성 시 중복된 주민등록번호와 기등록된 주민등록번호를 입력할 경우 해당 회원은 등록 되지 않습니다.

※ 정보 갱신(업데이트) 진행 단계

  • 회원정보 조회를 통해 업데이트 희망 회원 목록을 엑셀 파일로 다운로드
  • 다운로드 받은 파일을 확인하여 변경 희망항목에 대한 정보 수정 후 저장
  • 수정 후 저장한 파일을 활용하여 정보 갱신(업데이트) 진행

※ 엑셀 파일 업로드 유의사항

  • 반드시 회원정보 목록에서 파일을 다운로드 받은 후 활용하세요
  • 정보 갱신을 위한 회원 비교를 위해 SEQ, 아이디는 변경할 수 없습니다.
  • SEQ, 아이디 정보가 일치하지 않는 경우 해당 정보는 갱신되지 않습니다.
